# Project1_CineVerse
CineVerse is a responsive movie streaming front-end website built using HTML, CSS, and JavaScript.
It recreates the look and feel of modern streaming platforms with smooth UI interactions and an engaging layout.


This project focuses on building a visually appealing, interactive, and fully responsive movie streaming interface using pure front-end technologies.
HTML5 – Structure
CSS3 – Styling & Layout
JavaScript (Vanilla JS) – Interactivity
